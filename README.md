# Smart Power Quality Analyzer

## Overview

Real-time power quality monitoring system designed to measure RMS voltage, RMS current, power factor, frequency, and harmonic distortion (THD) for industrial energy monitoring applications.

## Features

* Analog signal acquisition
* RMS voltage and current measurement
* Power factor calculation
* Frequency analysis
* Harmonic distortion (THD) measurement
* FFT-based signal processing
* Wireless data transmission
* Remote monitoring through a web application

## Technologies
